# SingleTape — Android TWA Build Guide

This guide walks you through building the Android App Bundle (.aab) for Google Play using **Bubblewrap CLI**.

---

## Prerequisites

| Tool | Version | Install |
|------|---------|---------|
| Node.js | 18+ | https://nodejs.org |
| JDK | 17 | `sudo apt install openjdk-17-jdk` or https://adoptium.net |
| Android SDK | API 34 | Installed automatically by Bubblewrap on first run |
